

## Objectives of a Good Firewall

- Resilient to attacks
- Traffic between networks must be forced through firewall


## Firewall Justifications

- Risks
	- Exposure to sensitive systems
	- Exploitation of protocol flaws
	- Unauthorized users
	- Malicious data
- Limitations
	- Configuration mistakes have serious consequences
	- Not all network applications were written to survive going through a firewall
	- Individuals who are forced to use a firewall might engineer a way around
	- Latency being added by the firewall



## Firewall Types

- Static Packet Filtering
- Application Layer Gateway
- Stateful Packet Filtering
- Application Inspection
- Transparent Firewalls
- Next-Generation Firewalls
